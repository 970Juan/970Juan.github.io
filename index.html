Lumvian

Aprende visualmente sobre puentes levadizos

Interactividad, animaciones y explicaciones sencillas pensadas para estudiantes y docentes.

Descripción del Proyecto

El proyecto consiste en la creación de una página web interactiva que exponga, de manera clara y visual, el funcionamiento de un puente levadizo. A través de ilustraciones, animaciones y explicaciones textuales, el sitio busca fomentar el aprendizaje y la curiosidad de los usuarios.

Objetivos

General: Diseñar e implementar una página web interactiva que explique el funcionamiento de un puente levadizo, sus características y su importancia, con fines educativos y de divulgación.

Específicos

  • Diseñar una interfaz web atractiva y accesible.
  • Implementar secciones informativas sobre historia, partes y funcionamiento.
  • Crear animaciones del mecanismo de elevación.
  • Integrar ejemplos de puentes levadizos reconocidos.
  • Evaluar la usabilidad mediante pruebas.

Público Objetivo

Dirigido a estudiantes, docentes y público general interesado en ingeniería civil y arquitectura. Edad objetivo: 12-40 años.

Características Principales

Lumvian combina texto, imagen y movimiento para facilitar la comprensión de procesos complejos relacionados con los puentes levadizos.

Tecnologías Utilizadas

HTML para la estructura, CSS para estilos y JavaScript para interacción y animaciones.

Diseño de la Interfaz (UI)

Interfaz accesible, responsiva y pensada en la experiencia del usuario. Incluye secciones explicativas y una animación interactiva del puente.

Referencias

  • Admin. (2023). Puente levadizo: Definición, Diseño y Tipos. Infraestructura Vial.
  • MDN. HTML: Lenguaje de etiquetas de hipertexto.
  • Equipo editorial, Etecé. (2025). Página web - Qué es, elementos, usos y tipos.
